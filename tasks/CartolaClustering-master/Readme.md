# Clustering

An study on Clustering based on datas of Cartola FC: https://github.com/henriquepgomide/caRtola

- Techiques:
	- KMeans
	- Principal Component Analisis
	- DecisionTrees
	- Random Forest
	
- Task from an Course of MachineLearning (EEC1509)

- Members:
	- Dhiego Souto
	- Patrick Terrematte
